## branch 새로 만들기

1. ```git branch 브랜치이름```
    > 브랜치 생성

2. ```git checkout 브랜치이름```
    > 브랜치 전환

3. ```git push origin 브랜치이름```
    > 원격 저장소에 반영하기

4. ```1, 2 를 동시에 진행```
    
    ```git checkout -b 브랜치이름```
